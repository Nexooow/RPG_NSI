# RPG_NSI

## fonctionnalitées

- déplacements entre sommets, chaque déplacement a un "poids" qui représente du temps, plus le temps est élevé, plus de chance de déclencher une ou plusieurs événements
- chaque événement;
  l'utilisateur peut réaliser des actions, qui peuvent lui apporter des effets positifs ou négatifs (voir déclencher des combats)
- combat tour par tour
